# useVisible

获取当前页面是否隐藏或处于后台

## 何时使用

当需要根据页面显隐进行判断

## API

```jsx | pure
const visible: Ref<boolean> = useVisible();
```

## 返回值说明

| 返回值  | 说明                 | 类型      |
| ------- | -------------------- | --------- |
| visible | 当前应用是否处于后台 | `Ref<boolean>` |
