# useModal

显示模态对话框

## 何时使用

当需要使用模态对话框

## API

```jsx | pure
const [show] = useModal(initialOption);
```

## 参数说明

| 参数          | 说明                                       | 类型                 | 默认值 |
| ------------- | ------------------------------------------ | -------------------- | ------ |
| initialOption | 初始提示框配置(若指定后面可与新的配置合并) | `General.IAnyObject` | -      |

## 返回值说明

| 返回值 | 说明           | 类型                                             |
| ------ | -------------- | ------------------------------------------------ |
| show   | 显示消息提示框 | `(options?: General.IAnyObject) => Promise<any>` |
