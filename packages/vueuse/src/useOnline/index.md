# useOnline

获取网络是否可用

## 何时使用

当需要根据当前网络状态做判断时

## API

```jsx | pure
const online: ComputedRef<boolean> = useOnline();
```

## 返回值说明

| 返回值 | 说明         | 类型      |
| ------ | ------------ | --------- |
| online | 当前网络状态 | `ComputedRef<boolean>` |
